# Mannat Kumar - Portfolio

A modern, responsive personal portfolio website showcasing my skills, projects, and educational journey as a Django Full Stack Developer. Built using HTML, Custom CSS, and Vanilla JavaScript.

## 🚀 Features

- **Modern UI/UX Design**: Clean, minimalist aesthetic utilizing glassmorphism styling and custom animations.
- **Dark/Light Theme Toggle**: Seamless style switching to accommodate user preferences.
- **Responsive Layout**: Fully responsive interface that adapts to desktop, tablet, and mobile devices.
- **Featured Projects**: Highlights key full-stack Django projects including *Shahiledger*, *formX*, and *gaudiango*.
- **Dynamic Certifications Grid**: Premium, interactive cards verifying relevant industry knowledge and coursework.

## 🛠️ Built With

- **HTML5**: Semantic markup emphasizing accessibility and SEO best practices.
- **CSS3**: Native CSS utilizing Flexbox, CSS Grid, custom variables (tokens), gradients, and keyframe micro-animations.
- **JavaScript (ES6+)**: Handles interactivity such as theme toggling, mobile navigation hamburger menus, and `IntersectionObserver` scroll reveal aesthetics.

## 📂 Repository Structure

- `index.html`: The central entry point and markup structure.
- `style.css`: All styling parameters, animations, and responsive breakpoints.
- `script.js`: Component logic.
- `myphoto1.jpg`: The hero profile asset.
- `favicon.svg`: Website icon.

## 🏃 Getting Started

To view this portfolio locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/mannatxdev/portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Open `index.html` in your preferred web browser, or use an extension like VS Code's **Live Server** for hot-reloading.
